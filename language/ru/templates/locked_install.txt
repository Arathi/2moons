<h2>Установщик заблокирован!</h2>
<p>Для включения мастера установки игры должен быть создан файл «ENABLE_INSTALL_TOOL».</p>
<ul>
    <li>В папке includes/ создайте файл с названием «ENABLE_INSTALL_TOOL». Имя файла должно быть в верхнем регистре. Сам файл должен быть без расширения и может быть пустым.</li>
</ul>
<p>В целях безопасности настоятельно рекомендуется переименовать файл или удалить его, когда установка будет завершена. Для надёжности можно удалить всю папку «install».</p>
